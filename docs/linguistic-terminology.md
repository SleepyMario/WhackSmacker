# Linguistic Terminology

WhackSmacker displays linguistic terminology from the standalone canonical repository:

```text
/home/ashwin/Projects/languages/linguistic-terminology
```

The terminology repository remains the source of truth. It is not a language curriculum and is not WhackSmacker application source code.

WhackSmacker uses a checked-in snapshot generated by:

```bash
npm run sync-linguistic-terminology
```

The synchronization script reads categorized Markdown term files, validates required fields and unique stable IDs, and writes a bundled TypeScript snapshot for runtime use. Packaged WhackSmacker reads only the bundled snapshot. It does not require the sibling terminology repository, GitHub, a network connection, or a Git submodule at runtime.
