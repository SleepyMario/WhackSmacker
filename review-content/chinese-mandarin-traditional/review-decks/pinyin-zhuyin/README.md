# Pinyin-Zhuyin Review Deck

Deck name: `Pinyin-Zhuyin`

This deck reviews conversion between 漢語拼音 and 注音符號 without tones.

Scope:

- Mandarin initials.
- Mandarin finals and component spellings.
- Apical or empty-rime syllables.
- Common standalone spelling conventions with `y` and `w`.
- No tone marks.
- No word vocabulary, grammar, Chinese-character vocabulary, or sentence cards.

The TSV file is bidirectional:

- Pinyin -> Zhuyin
- Zhuyin -> Pinyin

This is a simple review deck and input-system bridge, not a lesson sequence.
