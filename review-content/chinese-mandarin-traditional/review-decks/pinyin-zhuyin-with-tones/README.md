# Pinyin-Zhuyin with Tones Review Deck

Deck name: `Pinyin-Zhuyin with Tones`

This deck reviews tone-marked conversion between 漢語拼音 and 注音符號.

Scope:

- Tone marker conventions.
- Representative tone-marked syllables across Mandarin initials, finals, standalone spellings, ü/y-series, and apical or empty-rime syllables.
- Tone-marked Pinyin to tone-marked Zhuyin.
- Tone-marked Zhuyin to tone-marked Pinyin.
- No word vocabulary, grammar, Chinese-character vocabulary, or sentence cards.

Tone policy:

- First tone uses a marked Pinyin vowel and no Zhuyin tone mark.
- Second tone uses `ˊ` after the Zhuyin syllable.
- Third tone uses `ˇ` after the Zhuyin syllable.
- Fourth tone uses `ˋ` after the Zhuyin syllable.
- Neutral tone uses `˙` before the Zhuyin syllable.

The TSV file is bidirectional:

- Pinyin -> Zhuyin
- Zhuyin -> Pinyin

This is a simple review deck and input-system bridge, not a lesson sequence.
