# Vietnamese Core Chapters 1–5 Review Deck

Deck ID: `vi-core-review-001-005`

This is the one authoritative review deck for Vietnamese Core Chapters 1–5.
Its 64 schema-v2 vocabulary cards provide two directions for each of the 32 canonical
lexical items or senses first introduced in Chapters 1–5. There is no fixed
per-chapter or per-deck quota.

Each sense has a Vietnamese prompt/English answer card and an English prompt/Vietnamese answer card. Every card has no distractors and
uses one to three literal learner-facing dialogue or narrative sentences from
the item's true first-introduction chapter, with multiple distinct examples
where available.

The stable card ID identifies a card across releases. The generated pedagogical fingerprint changes only when pedagogically material content changes; provenance, ordering, package version, timestamps, and other metadata do not affect it.

No Foundation or Chapter 6+ material is required.
