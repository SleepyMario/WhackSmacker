# Dutch Core Reviews

SPDX-License-Identifier: GPL-3.0-or-later

This directory is the authoritative WhackSmacker source for Dutch review decks. The Dutch curriculum repository owns chapters, translations, ledgers, and paired Grammar Easy/Hard summaries; it does not duplicate these card files.

The complete consecutive five-chapter inventory is:

- Chapters 1-5: 35 senses / 70 bidirectional cards;
- Chapters 6-10: 40 senses / 80 bidirectional cards;
- Chapters 11-15: 33 senses / 66 bidirectional cards;
- Chapters 16-20: 42 senses / 84 bidirectional cards;
- Chapters 21-25: 40 senses / 80 bidirectional cards.

Every deck uses review schema v2, contains vocabulary/function-word cards only, and requires one to three literal examples from primary Dialogue or Narrative content.
